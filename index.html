Website

Um site sem hyper-text, estilos, e botões de interação não é um site. Por isso é necessário de conhecimentos em HTML 5, CSS 3 e JavaScript, para que um site bonito, informativo e dinâmico agrade os visitantes da página.

API

As API's, são de grande utilidade para facilitar a vida dos usuários de uma determinada aplicação. Ela auxilia conexões com outras aplicações e em geral trazem mais praticidade em uma série de processos.

Banco de Dados

O banco de dados mais utilizado é o gigante e famoso MySQL. Que por consequência possui algumas vantagens como código aberto, fácil de manipular, compatibilidade, suporte da comunidade e até mesmo seguraça.
