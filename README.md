# Portfolio

Portfólio pessoal onde mostro meus projetos, experiências e trajetórias.

O site foi inteiro desenvolvido em Angular e Tailwind, com desenvolvimento e arquitetura moderna.

Possui responsividade, modo light e modo dark, tradução para português e inglês.

É um site de página única, onde conto em detalhes sobre mim, meus projetos e especialidades.

Possui o tema em modo Dark e modo Light para escolha do usuário
